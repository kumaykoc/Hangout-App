🎉 Hangout App

Lefke'nin sosyal yaşamını keşfet — kafeler, barlar, özel teklifler ve etkinlikler tek bir yerde.

📱 Hakkında
Hangout App, Kuzey Kıbrıs'ın başkenti Lefkoşa'daki sosyal yaşamı dijitalleştirmek amacıyla geliştirilmiş bir mobil uygulamadır. Kullanıcılar şehirdeki kafe ve barlara özel tekliflere ulaşabilir, etkinlik biletlerini satın alabilir ve sosyal hayatlarını planlayabilir.

⚠️ Bu proje şu an aktif geliştirme aşamasında değildir. Kod tabanı yarım kalmış bir prototiptir.

✨ Özellikler

🗺️ Lefke'deki kafe ve barları keşfet
🎟️ Etkinlik bileti satın alma
🏷️ Mekânlara özel indirim ve teklifler
📅 Sosyal etkinlik takvimi

🛠️ Teknolojiler

Frontend: React Native, TypeScript
Platform: iOS & Android (Mobil)

🚀 Kurulum
bash# Repoyu klonla
git clone https://github.com/kumaykoc/Hangout-App.git

# Proje klasörüne gir
cd Hangout-App

# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npm start
👥 Geliştirici
Geliştirilen bu proje iki kişilik bir ekip tarafından başlatılmıştır.
